# 백앤드 평가

## 본 소스를 클론해서 양파마켓 백앤드 서버를 완성하시오. 아래의 기능이 사용가능해야함

---

- 회원가입
- 로그인 => 로그인시 json web token 발행
- 판매등록 => 등록시 jwt인증
- 판매목록 보기 => jwt인증
- 판매정보의 일련번호로 세부정보 판매정보 열람 => jwt인증

---
